This is a client and firmware distribution for the GoodFET, an
open source adapter for the MSP430, CC2430, SPI Flash, and
various other chips.

You should probably be using the latest code from SVN, which you
can grab like so:

svn co https://goodfet.svn.sourceforge.net/svnroot/goodfet
