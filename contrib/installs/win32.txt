Use TortoiseSVN to extract the following URL to C:\GOODFET\
https://goodfet.svn.sourceforge.net/svnroot/goodfet

Install the FTDI VCP Driver.  (Not the D2XX Direct Driver.)
http://www.ftdichip.com/Drivers/VCP.htm

These are handy for making Windows less angry.
http://console.sf.net/


Install GNUWin32
http://sourceforge.net/projects/getgnuwin32/files/


cd getgnuwin32
download
install c:\gnuwin32
Append "C:\gnuwin32\bin" to %PATH% in "My Computer/Properties/Advanced/Environment Variables"

As PySerial is presently broke in 64-bit Windows, be sure to install
the 32-bit build of it and all related Python tools.

Install Python 2.x
http://www.python.org/getit/
Append "C:\Python27" to %PATH%

Install pyserial for Python 2.x,
http://pypi.python.org/pypi/pyserial

Install Py2Exe for distributions.  This MUST match your Python version.
http://sourceforge.net/projects/py2exe/files/
